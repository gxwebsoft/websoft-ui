基于Vue3+Nuxt构建的现代化web前端框架，支持私有化部署，支持配置独立的云存储、支付系统、短信、微信小程序等，适用于构建企业官网、商城、办公系统等。
## 技术栈

- Vue 渐进式 JavaScript 框架
- Nuxt 适用于有SEO需求的web框架
- Node JavaScript 运行环境(建议版本18.x.x以上)
- Yarn 包管理工具
- Vite 下一代前端开发与构建工具
- ES6 ECMAScript 6
- And more...
